# PCA-KMEANS
K-Means is used to group customers into distinct clusters, while PCA helps reduce dimensionality and improve visualization. The model demonstrates effective customer segmentation, which can be used for targeted marketing and business decision-making.

The project applies  K-Means clustering with Principal Component Analysis (PCA) on Mall Customers dataset to segment customers based on their purchasing behavior .The features Annual Income and Spending Score arescaled and using PCA to visulize customer clusters effectively.
